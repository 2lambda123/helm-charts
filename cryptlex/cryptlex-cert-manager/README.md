# Cryptlex Cert-Manager Helm Chart

## Installing the Chart
You can install the Chart via Helm CLI:

```console
$ helm upgrade --install cryptlex-cert-manager --values <values file> --namespace cryptlex --version <version> cryptlex/cryptlex-cert-manager
```

## Uninstalling the Chart
You can uninstall the Chart via Helm CLI:

```console
$ helm uninstall cryptlex-cert-manager
```
